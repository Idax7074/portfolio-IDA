Tema 4 - Animation

04.04.01 Temaopgave

I opgave 04.04.01 temaopgave skulle vi designe og programmere et interaktivt og animeret spil. Vi skulle lave vores egne figurer og baggrunde i illustrator og gemme dem som svg filer. I vores spil skulle man kunne klikke på ting, som enten faldt ned eller dukkede op. Der skulle både være gode og dårlige ting at klikke på, hvor man enten fik point eller mistede og mistede liv. Derudover skulle der vises tid, så brugeren viste hvor lang tid der var tilbage. Vi skulle lave to slutninger en ”level complete” og en ”game over”.

Processen

Til denne opgave lærte vi om sproget Javascript, CSS animationer, aktivitetsdiagram og state machine diagram. Vi lærte om forskellige karakterdesign som line of action, hvor elementerne har mere dynamik og attitude. Ved at bruge former og linjer kan man understrege et elements personlighed. Vi lavede aktivitetsdiagram og state machine diagram, som er en forståelig model af vores spil.

Med CSS animationer kan man bestemmer animationens navn, hvor lang tid og hvor mange gange animationen skal forløbe. Vi lærte om keyframes, som er hvordan animation går fra en tilstand til en anden. Vi skulle opbygge vores spil via container og sprite princippet, ved at et spilelement i form af svg fil indsættes som baggrund på en sprite div, som placeres inden i en container div. På den måde ville vi både kunne animere container og sprite hver for sig.

I Javascript lærte vi om document.querySelector, funktioner, addClass/removeClass, variabler, events, betingelser og math.random. Så vi kunne klikke på vores elementer, få elementer til at placere sig forskellige, tælle point og liv.
