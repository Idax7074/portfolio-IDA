Tema 2 - Grundlæggende web

02.01.02 Website

I dette forløb skulle vi udarbejde vores første simple website. Ved at bruge grundlæggende html og css, som vi lærte i forløbet. I denne opgave skulle vi følge det givne layoutdiagram, wireframe, tekstdokument og bruge de vedlagte billeder. Vi skulle tilføje farver og to forskellige fonte. Vi skulle få sitet til og være responsivt og det skulle det validere uden nogen fejl.

Processen

Vi lærte at opsætte en HTML side, ved brug af semantiske tags, som vores h1, h2, og p. Vi brugte også html-attributter, som href for at kunne linke til html filerne på sitet. Vi fik fortalt hvordan vi ved brug af CSS kan ændre websitets baggrundsfarve, skriftfarve og skriftstørrelse. Desuden importerede vi vores egen font med font-face. Udover dette lærte vi om box-modellen, hvordan man med content, padding, border og margin kan få mellemrum rundt om og inde i boksen.

For at kunne få sitet til og være responsivt, skulle vi bruge media query, for at kunne bestemme hvornår forskellige CSS styles for mobilsitet skulle tilføjes. Det gjorde vi ved at bruge max-width.

Vi lærte om grid og flex-box, så kunne vi følge layoutdiagrammet. CSS grid er et todimensionelt layout, grid-elementerne er inddelt i kolonner og rækker. Med CSS flexbox kan du justere elementer lodret og vandret ved hjælp af kolonner og rækker. Elementerne vil flekse til forskellige størrelser for at fylde rummet.

02.02.01 Emnesite

I denne opgave skulle vi udarbejde moodboards, wireframe og styletile, som passede til den stil vi havde valgt. Ud fra stilen skulle vi bruge farver og typografi på vores site, der passede til stilen. Derudover skulle vi sørge for at vores site var respontivt og validerede.

Processen

I temaet lærte vi om forskellige design principper, som gestalt lovene, kontrast, typografi, farver. Vi skulle også lave og følge vores eget layoutdiagram. Jeg valgte at lave et opskriftside. Jeg ville gerne skabe et roligt og overskueligt site til unge, jeg valgte derfor en læsevenlig typografi "pontanosans". Mine farver på sitet var lyse og rolige. Disse er design principper, som bliver brugt i min valgte stilart, som er moderne flat design.

Kontrasten på mit site blev skabt ved mine store og fængede billeder, det var der øjet faldt til, så læseren hurtigt viste hvad opskriften omhandlede. Design principperne gør at sitet følger gestaltlovene, da det skaber et synligt hierarki, så du kan se hvad der er vigtigt.
